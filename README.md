📌 Sobre o Projeto- equipe: Evelyn Maida,Evelin Kailanny,Lilian Vitoria,Karina Cirilo, Anna Sophya.

Este projeto consiste no desenvolvimento de um site para uma jovem empreendedora que iniciou no ramo de vendas de trufas e bolos. O objetivo principal é facilitar o processo de compra para os clientes, além de auxiliar na organização dos pedidos e da rotina de trabalho da empreendedora.

O sistema busca oferecer uma experiência simples, prática e intuitiva, permitindo que os clientes visualizem os produtos disponíveis, realizem pedidos de forma rápida e acompanhem as informações necessárias para a compra.

🎯 Objetivo

Desenvolver uma plataforma web que permita:

Divulgar os produtos disponíveis;
Facilitar a realização de pedidos;
Organizar as vendas realizadas;
Melhorar o atendimento ao cliente;
Otimizar a gestão do pequeno negócio.

👥 Público-alvo
Clientes interessados em comprar trufas e bolos. No geral, coisas doces.
A empreendedora responsável pelas vendas e gerenciamento dos pedidos.

✨ Funcionalidades

Para os clientes

Visualizar o catálogo de produtos;

Consultar preços;

Ver descrições e imagens dos produtos;

Fazer pedidos online;

Informar dados para entrega ou retirada;

Entrar em contato com a vendedora.
Para a administradora
Cadastrar novos produtos;
Editar ou remover produtos;
Atualizar preços;
Organizar pedidos recebidos;
Gerenciar estoque básico;
Acompanhar as vendas.
